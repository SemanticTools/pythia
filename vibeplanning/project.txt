This is a technical project about making a knowledge base available / editable with LLM assistence

Code: nodejs
Type of coding: mjs files, modules, not classes
Very low on dependencies.  If we can do ourselves in <50 lines, we do that instead.
Nothing is hardcoded. We have our system.toml file to write all our settings.

Split functionalities clean, files should solve one problem only (ie, logging, storage, webserver, etc.... )

##The system to build.

We want to be able to upload very many text pages, and index them by words.
Indexes converted to embeddings.
Then we want to be able to ask questions about it.
Prompt is converted to embeddings.
Embeddings used to search close by embeddings. top x pages will be preloaded into the prompt.
The question will then be posted to LLM.
